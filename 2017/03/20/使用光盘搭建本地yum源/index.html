使用光盘搭建本地yum源

2017-03-20 
适用场景： 刚装好的系统，想安装一些常用软件和一些包组的时候，就可以使用安装光盘搭建本地yum

第一步：挂载安装光盘

mount /dev/cdrom /mnt

第二步：编辑repo yum源文件

cd /etc/yum.repos.d/
mkdir backup
mv *.repo backup/ #备份原repo文件

[root@localhost yum.repos.d]# vi local.repo
[local]    #yum仓库的名字,被yum命令识别
name=local             #这个name随便填
baseurl=file:///mnt    #下载地址，这里我们用的file://协议, 而/mnt是之前光盘挂载路径
gpgcheck=0             #不使用gpg检查rpm包的来源
enabled=1              #这个选项表示启动repo源，设置为0的时候，表示不启动

第三步：检查

yum makecache

第四步: 测试效果

yum install php -y
