CentOS7 安装及破解confluence

2017-10-17

confluence是什么？

简单说：它是一款卓越的团队协作软件,改变了现代团队的工作方式

环境准备

安装配置Java

官方：http://www.oracle.com/technetwork/cn/java/javase/downloads/jdk8-downloads-2133151-zhs.html

(1)下载jdk-8u144-linux-x64.tar.gz二进制包
wget http://download.oracle.com/otn-pub/java/jdk/8u144-b01/090f390dda5b47b9b721c7dfaa008135/jdk-8u144-linux-x64.tar.gz?AuthParam=1508210153_b10ed029a2d26362f2bc3c77cd760dc4

(2)解压至/usr/local/web目录
   mkdir /usr/local/web -p
   tar -xf jdk-8u144-linux-x64.tar.gz 
   mv jdk1.8.0_144 /usr/local/web/

(3)配置环境变量
JAVA_HOME=/usr/local/web/jdk1.8.0_144
PATH=$JAVA_HOME/bin:$PATH:.
CLASSPATH=$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:.
export JAVA_HOME
export PATH
export CLASSPATH

(4)查看java版本
[root@localhost ~]# java -version
java version "1.8.0_144"
Java(TM) SE Runtime Environment (build 1.8.0_144-b01)
Java HotSpot(TM) 64-Bit Server VM (build 25.144-b01, mixed mode)

安装mysql-二进制

各版本下载源：http://mirrors.sohu.com/mysql/MySQL-5.6/    #不太好找的话搜索：“glibc2.5-x86_64”即可

(1)下载mysql-5.6.35-linux-glibc2.5-x86_64.tar.gz
   http://mirrors.sohu.com/mysql/MySQL-5.6/mysql-5.6.35-linux-glibc2.5-x86_64.tar.gz?crazycache=1

(2) 添加mysql程序运行时的管理用户
    useradd -s /sbin/nologin  -M mysql

(3)将解压后的程序目录放到/usr/local/web目录，并进行软链为mysql
   tar -zxvf mysql-5.6.35-linux-glibc2.5-x86_64.tar.gz
   mv mysql-5.6.35-linux-glibc2.5-x86_64 /usr/local/web/
   cd /usr/local/web
   ln -s mysql-5.6.35-linux-glibc2.5-x86_64 mysql

(4)修改权限
   cd mysql && chown -R mysql.mysql data

(5)初始化数据库
   /usr/local/web/mysql/scripts/mysql_install_db --basedir=/usr/local/web/mysql --datadir=/usr/local/web/mysql/data --user=mysql

(6)拷贝脚本文件到/etc/init.d/目录
   cp /usr/local/web/mysql/support-files/mysql.server /etc/init.d/mysqld
   ll /etc/init.d/mysqld

(7) 修改启动脚本和mysql命令的路径（默认/usr/local/目录下）
    sed -i 's#/usr/local/mysql#/usr/local/web/mysql#g' /usr/local/web/mysql/bin/mysqld_safe /etc/init.d/mysqld

(8) 复制默认配置文件
    \cp /usr/local/web/mysql/support-files/my-default.cnf /etc/my.cnf 

(9) 启动mysql并配置开机自启
    /etc/init.d/mysqld start
    chkconfig --add mysqld
    chkconfig mysqld on
    chkconfig --list

(10)配置mysql环境变量
    echo 'export PATH=/usr/local/web/mysql/bin:$PATH' >>/etc/profile
    source /etc/profile
    which mysql
(11)设置密码
    /usr/local/web/mysql/bin/mysqladmin -u root password '123456'
    mysql -uroot -p

(12)创建名为confluence的数据库并赋权
    create database confluence default character set utf8 collate utf8_bin;
    grant all on `confluence`.* to 'root'@'%' identified by '123456';
    flush  privileges;

下载安装confluence5.6.6

官方下载：https://www.atlassian.com/software/confluence/download-archives

下载：atlassian-confluence-5.6.6-x64.bin
(1)添加权限：
   chmod 755 atlassian-confluence-5.6.6-x64.bin
(2)执行bin来安装
   ./atlassian-confluence-5.6.6-x64.bin

  根据提示指定安装目录、端口等信息
  Confluence 安装目录/usr/local/web/confluence/confluence
  Confluence data目录: /usr/local/web/confluence/confluence_data
  HTTP: 8090, Control: 8000

(3)安装后的信息
   (1)修改端口：vim /usr/local/web/confluence/confluence/conf/server.xml
   (2)wiki 默认端口是8090  需要在防火墙开放8090端口

破解confluence

(1)停止confluence服务
   /etc/init.d/confluence stop

(2)下载破解程序
   confluence5.6.6-crack.zip   百度云：https://yun.baidu.com/s/1o88rrIU

(3)删除confluence程序目录下的6个jar包
   cd /usr/local/web/confluence/confluence/confluence/WEB-INF/lib

   要删除的6个jar包名称分别为：
           atlassian-extras-api-3.2.jar
           atlassian-extras-common-3.2.jar
           atlassian-extras-core-3.2.jar
           atlassian-extras-decoder-api-3.2.jar
           atlassian-extras-decoder-v2-3.2.jar
           atlassian-extras-legacy-3.2.jar
   为避免不必要的麻烦，把这6个jar文件移动到该目录新建的bak目录下

(4)解压破解程序，把jar子目录里面的三个jar包，放到/usr/local/web/confluence/confluence/confluence/WEB-INF/lib目录下

  cp atlassian-extras-3.2.jar /usr/local/web/confluence/confluence/confluence/WEB-INF/lib/

  cp Confluence-5.6.6-language-pack-zh_CN.jar /usr/local/web/confluence/confluence/confluence/WEB-INF/lib/

  cp mysql-connector-java-5.1.39-bin.jar /usr/local/web/confluence/confluence/confluence/WEB-INF/lib/

Web配置

(1)Web配置
   启动服务：/etc/init.d/confluence start
   访问：http://server_ip:8090
   选择“Production Installation”方式-->Start setup进入输入激活码页面

   好，接下来在windows上操作(前提：java环境配置好)
   打开cmd--进入解压后的confluence5.6.6-crack目录，执行命令:java -jar confluence_keygen.jar,在弹出的窗口中输入Web页面的ServerID等信息获取License Key,最后粘贴到Web页面中。

(2)选择Mysql数据库-- External Database -- Direct JDBC  -- 输入mysql账号和密码 -- Next

注：如果安装的是MySQL5.7,此处会出现“InnoDB”错误，需要在/etc/my.ini文件中添加一行，然后重启mysql服务
添加：default-storage-engine=InnoDB

(3)然后选择“Empty Site”  -- Manage Users and groups within Confluence  -- 新建Web管理员账户 -- 完成

用户前台页面：http://Server-IP:8090/dashboard.action
管理员控制台：http://Server-IP:8090/admin/console.action

故障处理

创建空间后发现有中文乱码，修改下mysql的配置文件即可。
[root@localhost web2016]#vi  /etc/my.cnf
[client]
default-character-set=utf8

[mysqld]
character-set-server=utf8

重启mysqld
/etc/init.d/mysqld restart
查看mysql字符集是否已经更正
> show variables like 'character_set%';
