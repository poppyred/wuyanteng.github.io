定义变量的三种方式

  • 直接赋值
如：x=10 或 file=/etc/hosts

规律：脚本中能用变量的地方尽量使用变量
  • 传参(传递参数)
$0 $1 ... $n  $#

传参方式让脚本更加灵活
  • 交互式设置变量
使用read命令实现脚本交互。 

[root@jkbackup scripts]# read
oldboy                 #输入字符就相当于给$REPLY变量赋值
[root@jkbackup scripts]# echo $REPLY
oldboy             

[root@jkbackup scripts]# read -p "请输入你的年龄:"
请输入你的年龄:18
[root@jkbackup scripts]# echo $REPLY
18

-p 设置提示语句
age 设置的变量名称,前面要有空格

把值明确赋值给某个变量

[root@jkbackup scripts]# read -p "请输入你的姓名:" age
请输入你的姓名:garywu
[root@jkbackup scripts]# echo $age
garywu
[root@jkbackup scripts]# echo $REPLY     #可以看到$REPLY的值没变
18

交互输入银行卡账号密码

[root@jkbackup scripts]# cat test.sh 
read -t 5 -p "请输入你的银行卡账号: " account
read -t 5 -s -p "请输入你的银行卡密码: " pass
echo
echo "你的银行卡卡号：" $account ，"你的银行卡密码：" $pass 

注：-s 用来关闭回显,即不显示输入的内容
   -t  设置超时时间,单位:秒。超时后将退出交互
   echo  shell换行符

结果:
[root@jkbackup scripts]# sh test.sh 
请输入你的银行卡账号: 12345
请输入你的银行卡密码: 
你的银行卡卡号： 12345 ，你的银行卡密码： 67890

条件表达式

对环境等进行判断

文件判断

先敲一对[],然后退格输入2个空格[  ],最后再回退一个空格开始输入[ -f /etc/hosts ]

文件判断
[root@jkbackup scripts]# [  -f /etc/hosts ]
[root@jkbackup scripts]# echo $?       #0表示命令执行成功，文件存在
0

[ -f /etc/hosts ] && echo "表达式成立"  || echo "表达式不成立"
[ -f /etc/host ] && echo "表达式成立"  || echo "表达式不成立"
[ -f /etc/hosts ] && cat /etc/hosts  || echo "文件不存在"
[ -f /etc/host ] && cat /etc/hosts  || echo "文件不存在"

[ -d /etc1/ ] && echo "表达式成立"  || echo "表达式不成立"
表达式不成立
[ -d /etc/ ] && echo "表达式成立"  || echo "表达式不成立"
表达式成立

字符串判断

[root@jenkins scripts]# name=
[root@jenkins scripts]# [ -z "$name" ] && echo "表达式成立"  || echo "表达式不成立"
表达式成立
[root@jenkins scripts]# name=" "
[root@jenkins scripts]# [ -z "$name" ] && echo "表达式成立"  || echo "表达式不成立"
表达式不成立

[root@jenkins scripts]# [ -z "$age" ] && echo "表达式成立"  || echo "表达式不成立"
表达式成立
[root@jenkins scripts]# [ -n "$age" ] && echo "表达式成立"  || echo "表达式不成立"

[ "$name1" == "$name2" ] && echo "表达式成立"  || echo "表达式不成立"
[ "$name1" != "$name2" ] && echo "表达式成立"  || echo "表达式不成立"

整数判断

-eq     equal 等于
-ne     not equal 不等于
-gt     great than 大于
-ge     great equal 大于等于
-lt     less than   小于
-le     less equal  小于等于

[root@jkbackup scripts]# [ 1 -ne 2 ] && echo "表达式成立" || echo "表达式不成立"
表达式成立
[root@jkbackup scripts]# [ 10 -le 2 ] && echo "表达式成立" || echo "表达式不成 立"
表达式不成立
[root@jkbackup scripts]# [ 10 -gt 19 ] && echo "表达式成立" || echo "表达式不.立"
表达式不成立
[root@jkbackup scripts]# [ 10 -le 19 ] && echo "表达式成立" || echo "表达式不.立"
表达式成立

逻辑判断符

与    简写：-a
或    简写：-o
非    简写: ！

[root@jkbackup scripts]# [ 10 -le 19 -a 5 -gt 8 ] && echo "表达式成立" || echo "表达式不成立""
表达式不成立
[root@jkbackup scripts]# 
[root@jkbackup scripts]# [ 10 -le 19 -a 5 -lt 8 ] && echo "表达式成立" || echo "表达式不成立""
表达式成立
[root@jkbackup scripts]# [ 10 -le 19 -a ! 5 -lt 8 ] && echo "表达式成立" || echo "表达式不成立"
表达式不成立

判断两个数字-测试

直接赋值方式

[root@jkbackup shell]# cat read.sh 
#!/bin/bash

num1=10
num2=3

#判断变量不为空
[ -z "$num1" ] && echo "第一个变量未赋值" && exit 1
[ -z "$num2" ] && echo "第二个变量未赋值" && exit 1

#判断是不是整数
expr 2 + "$num1" &>/dev/null
[ $? -eq 2 ] && echo “第一个不是数字” && exit 1
expr 2 + "$num2" &>/dev/null
[ $? -eq 2 ] && echo “第二个不是数字” && exit 1


#正式判断
[ $num1 -gt $num2 ] && echo "$num1 > $num2" && exit
[ $num1 -eq $num2 ] && echo "$num1 = $num2" && exit
[ $num1 -lt $num2 ] && echo "$num1 < $num2"

条件语句

if条件语句

单分支语句

如果  条件成立；则
    执行命令    

if [ $num1 -gt $num2 ];then
   echo "$num1" > "$num2"
   exit
fi   

if [ $num1 -lt $num2 ];then
   echo "$num1" < "$num2"
   exit
fi   

if [ $num1 -eq $num2 ];then
   echo "$num1" = "$num2"
fi

双分支语句

if [ $num1 -gt $num2 ];then
   echo "$num1" > "$num2"
   exit
else 
    echo "$num1" 不大于 "$num2"

fi

多分支语句

if [ $num1 -gt $num2 ];then
   echo "$num1 > $num2"

elif  [ $num1 -eq $num2 ];then
    echo "$num1 = $num2"

else
     echo "$num1 < $num2"

fi

小结

单分支：一个条件一个结果
双分支：一个条件多个结果
多分支语句: 多个条件多个结果
