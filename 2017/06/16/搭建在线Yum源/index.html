    2017-06-16

    尽管有很多的免费镜像提供Yum源服务，但还是有必要建立自己的Yum源服务器，主要出于以下几点考虑：

    网络速度：访问互联网可能比较慢

    节省带宽：如果有大量的服务器，假设自己的yum源可以有效节省互联网带宽

    联网限制：对于有些内网服务器，不能连接到互联网

    Yum服务端部署

    编译安装Nginx

    1. 获取Nginxs软件包
   wget http://tengine.taobao.org/download/tengine-2.2.0.tar.gz

2. 解压安装包并进入安装文件夹
   tar xf tengine-2.2.0.tar.gz  && cd  tengine-2.2.0

3. 安装编译需要的依赖包
   yum install gcc openssl-devel pcre-devel zlib-devel -y

4. 编译安装Nginx
   ./configure && make && make install

5. 配置Nginx开机自启动
   cd /etc/init.d/
   cat >> nginx <<EOF
        #!/bin/bash
        #chkconfig: 2345 70 30
        #description:                 
        #processname: nginx

        /usr/local/nginx/sbin/nginx

   chmod +x nginx
   chkconfig --add nginx
   chkconfig nginx on
   chkconfig --list |grep nginx

6. 配置Nginx文件
   vim /usr/local/nginx/conf/nginx.conf

          location / {
                root  /rpm;
                autoindex on;
                #root   html;
                #index  index.html index.htm;
          }

    配置欲同步的网络源

    1、备份CentOS 自带yum源
   mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup

2、改为国内aliyun yum源
   wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-6.repo

注：当然，此处也可以直接使用epel-realease源（推荐原因: 软件更齐全）
    yum install -y epel-realease

3、删除yum源缓存
   yum clear all

4、创建yum源缓存
   yum makecache

5、查看yum源repo列表
   yum repolist

    同步网络yum源

    1. 切换到网站根目录
    cd /usr/local/nginx/html/    
    mkdir ./rpm  && cd ./rpm   进入rpm网站根目录

2. 安装yum工具包
    yum -y install yum-utils

3. 同步网络yum源
   reposync -r base

4. 创建yum源的repo配置文件
   yum install -y createrepo
   createrepo  base/Packages/

    网页访问部署完成的yum源

    /usr/local/nginx/sbin/nginx

http://192.168.1.10      确认网页访问正常

    客户端配置yum源

    1. 备份CentOS自带yum源
    mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup

2. 拷贝并编辑Base
    cp /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak
    vim  /etc/yum.repos.d/CentOS-Base.repo

    修改baseurl为以下实际本地yum地址
    [base]
    baseurl=http://192.168.1.10/base/Packages/$basearch/

    [expel]
    baseurl=http://192.168.1.10/epel/$basearch/

    [updates]
    baseurl=http://192.168.1.10/updates/Packages/$basearch/  

注：上面的base 、expel和updates等需要有对应的目录。

3. 删除yum源缓存
   yum clean all

4. 创建yum源缓存
   yum makecache

5. 查看yum源repo列表
    yum repolist

    客户端yum安装软件测试

    yum install -y ntpdate
