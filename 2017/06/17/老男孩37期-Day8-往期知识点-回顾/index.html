sed
sed -n '/你要找的内容（正则）/p' oldboy.txt    //sed查找内容

查找-示例1：
sed -n '/^oldboy/p' oldboy.txt        -n和p 意思是临时显示内容（没有实际修改）

删除某行-示例：
sed '/你要找的内容/d'  oldboy.txt       d表示删除

替换-示例：
sed -n '/3306/s#3306#3389#gp' services     //3306端口改为3389

sed -n '/3306/p' services |sed 's#3306#3389#g'   //3306端口改为3389。

s和g组合表示全局替换（s替换，g表示全局）
注：-n禁止与-i一起使用，否则会造成源文件内容被清空

sed -i '/3306/s#3306#3389#gp' services     //正式写入文件
sed -n '/3389/p' services   //查看修改结果

取IP地址(跟大牛学命令)

awk取IP地址思路：
ifconfig eth0 |awk -F '[ :]+' '{print}'   打印所有
ifconfig eth0 |awk -F '[ :]+' '/inet addr/{print}' 打印inet addr的行
ifconfig eth0 |awk -F '[ :]+' '/inet addr/{print $4}'  取IP列

sed取IP
ifconfig eth0 |sed -rn 's###gp'   sed查找替换打印-标准格式
ifconfig eth0 |sed -rn 's#^.*inet addr:(.*) Bcast:.*#\1#gp'

测试题
1. 查找出/tmp目录下面修改时间是7天以前，大小在50k到2M之间，并以.log结尾的文件把这些文件复制到/data目录
方法一：
find /tmp/ -type f -mtime +7 -size +50k -size -2M -name "*.log"|xargs -i cp {} /data
默认xargs不支持花括号，加速-i参数就可以用{}花括号了。

方法二：
find /tmp/ -type f -mtime +7 -size +50k -size -2M -name "*.log" -exec cp {} /tmp \;
{}表示find命令找到的文件。

-mtime +7表示7天之前，7表示第7天，-7表示最近7天

方法三:
find /tmp/ -type f -mtime +7 -size +50k -size -2M -name "*.log"|xargs  cp -t /data

方法四：
cp $( find /oldboy/ -type f -name "*.log" -size +1M) /tmp/b

查找/oldboy目录下所有7天以前的，以log结尾的，并且大于1M的文件，把这些文件移动到/tmp下.
find /oldboy  -type f -size +1M -name "*.log" -mtime +7 | xargs -i mv {} /tmp/

2. 查找出/tmp目录下面修改时间是7天以前，大小在50k到2M之间，并以.log结尾的文件
   find /tmp/ -type f -mtime +7 -size +50k -size -2M -name "*.log" 
   这是个非常常用的参数-size根据文件大小查找文件
   -size +50k 表示 大于50k的文件
   -size -2M 表示  小于2M的文件
   在find命令中，默认就表示并且 -size +50k 并且 -size -2M 


3. 在上题的基础上，使用命令调换passwd文件里root位置和/bin/bash位置，即第一列和最后一列位置调换
方法1：
[root@oldboy202 ~]# head passwd |sed -r 's#(^.*)(:x.*:)(.*)#\3\2\1#g'
/bin/bash:x:0:0:root:/root:root
/sbin/nologin:x:1:1:bin:/bin:bin
/sbin/nologin:x:2:2:daemon:/sbin:daemon
/sbin/nologin:x:3:4:adm:/var/adm:adm

方法3：
awk -F ':' '{OFS=":";print}' /etc/passwd  固定格式
awk -F ':''{OFS=":";tmp=$1;$1=$7;$7=tmp;print}' /etc/passwd    
第一个冒号：是输入分割
第二个冒号：表示OFS输出分割字符
tmp可以理解为一个空箱子，先把$1的苹果装箱子，然后把$1的苹果放到$7香蕉的位置,最后$7的香蕉装箱子放到前面位置。

6. 如何把/data目录下所有包含oldboy的目录（可能目录的目录的目录里还有oldboy目录） 目录都打包出来。要求:解压打包后的目录结构不能改变。
   find /data/ -type d -name "oldboy" |xargs tar zcf data.tar.gz
   tar zcf data.tar.gz  ${find /data/ -type d -name "oldboy"}

   find /data/ -type d -name "oldboy" -exec tar zcf data.tar.gz {} \;
   注：-exec find命令找到一个文件就给后面的命令处理一次，而压缩名字一直没变，所以多次压缩结果均会被覆盖，最后压缩包里面只有1个最新被压缩的目录


   tar xf data.tar.gz  不解压直接看压缩包内容
