vmware虚拟机-Linux多网卡配置

2017-08-05

VMware虚拟机克隆与配置修改

1. 添加多网卡
2. 复制网卡配置文件并修改IP地址、子网掩码等信息
   cd /etc/sysconfig/network-scripts/
   cp ifcfg-eth0 ifcfg-eth1
   # 一清空
   >/etc/udev/rules.d/70-persistent-net.rules 
   #开机自动清空网卡规则
   echo '>/etc/udev/rules.d/70-persistent-net.rules' >>/etc/rc.local 
   tail -1 /etc/rc.local

   # 两删除
   grep -E "HWADDR|UUID" /etc/sysconfig/network-scripts/ifcfg-eth[01]
   sed -r "/HWADDR|UUID/d" /etc/sysconfig/network-scripts/ifcfg-eth[01]
   sed -ri "/HWADDR|UUID/d" /etc/sysconfig/network-scripts/ifcfg-eth[01]
   grep -E "HWADDR|UUID" /etc/sysconfig/network-scripts/ifcfg-eth[01]

4. 重启网卡
   /etc/init.d/network restart

注：Windows网络连接中vmnet1和vmnet8 IP地址不能与网关冲突

预览网卡配置信息
