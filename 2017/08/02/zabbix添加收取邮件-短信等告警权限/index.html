zabbix添加收取邮件/短信等告警权限

2017-08-02

需求：给同事开放收取某机房告警邮件/短信权限

步骤如下：

  1. 添加用户群组 –> 指定群组权限
  2. 添加用户–>添加对应群组（继承群组权限）–> 配置告警媒介（添加告警收件人邮箱）
  3. zabbix登陆该用户验证权限
  4. 组态–动作配置邮件模板等
