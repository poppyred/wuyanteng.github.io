Cacti网卡流量达到100M后-流量图不准问题

2017-08-30

因业务需要，某台机器流量上升到了100M，但是经过排查流量图显示有误，不准确。

Google后发现问题原因在于网卡监控参数配置不正确，当流量超过100M后，导致图会出现频繁“跳水”

解决方案：

1）删除之前的监控图以及对应图形树

2) 【重新添加】在Cacti管理页面中选择Console->Data Source, 找到需要修改的机器和对应端口（即流量大于100M的端口），修改Output Type ID为In/Out bits (64-bit counters)（原来为In/Out bits），修改Maximum Value 为 1000000000

注：如果监控图出现了多个eth0网卡监控图，需要删除此主机，然后按照上面方法重新添加
