IP地址进制转换
IP地址是十进制，系统只认二进制（8位,只用0或1来表示），所以需要学会计算

IP地址包括：网络位和主机位

特殊IP地址说明：

1. 主机位全为0的称为网络地址，比如:10.0.0.0   10.0.0.0就是网络地址
2. 主机位全为1的称为广播地址，即向所有人发出消息。比如：10.0.0.0的广播地址为10.255.255.255

一个局域网的主机个数=2的N次方-2

【N表示有多少个主机位； “-2”-意思是去掉一个网段地址和去掉一个广播地址】

三个主类地址的掩码信息：
    A类地址掩码: 255.0.0.0      /8
    B类地址掩码：255.255.0.0    /16
    C类地址掩码：255.255.255.0  /24

IP地址划分-技巧

速算口诀：
1    128
11   192
111  224
1111 240
11111 248
111111 252
1111111 254
11111111 255
###############################################
例1：172.16.0.0/14 ,求子网掩码与可用主机数量
(1)子网掩码计算：255.252.0.0 【一个255占用八位，不足8位查上面的口诀，剩余6位得出是252】
(2)IP地址可用范围计算，使用255.255.255.255减去子网掩码得出第二个八位是3
(3)即172.16.0.1 - 172.（16+3）.0.254. 
   最后结果是：172.16.0.1 - 172.19.0.254 【已经减去网络地址和广播地址】

例2：192.168.0.0/30
子网掩码为：255.255.255.252
可用IP地址范围：192.168.0.1-192.168.0.2

例3；192.168.0.0/27
子网掩码为：255.255.255.224
可用IP地址计算：192.168.0.1 - 192.168.0.30

例4：172.16.0.0/17
子网掩码为：255.255.128.0
可用IP地址范围：172.16.0.1 - 172.16.127.254

查看网关信息

1. route -n
2. netstat -rn
3. ip route

route更改
#删除网关：
[root@root ~]# route del default gw 192.168.100.2
#添加网关
[root@root ~]# route add default gw 192.168.100.2
#1台服务器向访问另一个子网，如何添加路由？
route add -net 172.16.1.0 network 255.255.255.0 gw 172.16.2.1
注：-net 指定目的子网 gw指定通过哪个网关到达
route add -net 172.16.1.0 network 255.255.255.0 dev tun0
route add -net 172.16.1.0/24 dev tun0

#永久添加路由
vim /etc/sysconfig/static-routes
any net 172.16.1.0/24 gw 172.16.1.0

IP别名与辅助IP配置
(1) 不建议常用
[root@root ~]# ifconfig eth0:0 192.168.100.101/24 up   配置别名IP，即启用虚拟IP
ifconfig eth0:0 down  删除IP别名
别名IP网卡重启也生效
#编辑配置文件使其生效
vim /etc/sysconfig/network-scripts/ifcfg-eth0:1

(2)辅助IP配置【常用】
#添加
[root@root ~]# ip addr add 192.168.100.101/24 dev eth0
[root@root ~]# ip addr add 192.168.100.102/24 dev eth0 label eth0:1
#删除
 [root@root ~]# ip addr del 192.168.100.102/24 dev eth0 label eth0:1
 [root@root ~]# ip addr del 192.168.100.101/24 dev eth0
注：label用来 添加辅助IP标签

Ping命令扩展
#共计ping 3次
ping -c 10 baidu.com

# 每次ping延迟5s 
ping -c 3 -i 5 www.baidu.com

# ping 5次不显示过程，直接给我结果
[root@root ~]# ping -c5 -q qq.com

# ping1000个包快速显示结果，无需等待
[root@root ~]# ping -c 1000 -f www.qq.com

统计访问服务器IP的ESTABLISHED连接数最多的IP？

[root@root ~]# awk -F "[ :]+" '/EST/{print $6}' netstat.log|sort|uniq -c|sort -rn -k1

tcpdump语法格式

-i 指定一个网卡
-nn 不需要将端口号转换为协议名称
-c  指定抓包次数
-w  保存抓包结果到指定文件
-X  指定某个端口

# 抓取53端口的数据包（待测试）
[root@root ~]# tcpdump -i eth0 -nn -X 'port 53' -c 10
# 抓包后追加到指定文件
[root@root ~]# tcpdump -i eth0 -c 10 -w /root/tcpdump.txt
#查看tcpdump抓包文件
[root@root ~]# tcpdump -r tcpdump.txt
