cobbler使用DTK自动化升级BIOS

2017-04-17 
cobbler使用DTK自动化升级BIOS

下载DTK ISO镜像

地址：https://www.dell.com/support/home/cn/zh/cndhs1/Drivers/DriversDetails?driverId=GYHHX
挂载iso并安装rpm基础包
mount -o loop dtk_5.3.0_1908_Linux64_A00.iso /mnt
cd /mnt/RPMs/rhel7/x86_64
ls        

#注:以下包安装有前后顺序
yum install -y srvadmin*
yum install -y raidcfg-5.3.0-1908.9058.el7.x86_64.rpm
yum install -y syscfg-5.3.0-1908.9058.el7.x86_64.rpm
yum install -y dtk-scripts-5.3.0-1908.9058.el7.x86_64.rpm

命令软链
ln -sv /opt/dell/toolkit/bin/raidcfg /usr/sbin/raidcfg

注：我的cobbler部署在了CentOS7上，所以需要进入rhel7目录。
了解文件SA.1和SA.2
# ls -lh /mnt/isolinux/
......
-r--r--r-- 1 root root 4.0M Feb 15  2016 SA.1
-r--r--r-- 1 root root 158M Feb 15  2016 SA.2
......

# file /mnt/isolinux/SA.1 
# file /mnt/isolinux/SA.2

上面可以看到有两个文件SA.1和SA.2, 这两个文件分别是kernel和initrd文件,稍后导入到cobbler的时候会用到它们。
手动添加dtk所需文件到cobbler启动菜单
mkdir -p /var/www/cobbler/ks_mirror/BIOS/R610
mkdir -p /var/www/cobbler/ks_mirror/BIOS/R620
mkdir -p /var/www/cobbler/ks_mirror/BIOS/R710
mkdir -p /var/www/cobbler/ks_mirror/BIOS/R720
cp /mnt/isolinux/SA* /var/www/cobbler/ks_mirror/BIOS/R610
cp /mnt/isolinux/SA* /var/www/cobbler/ks_mirror/BIOS/R620
cp /mnt/isolinux/SA* /var/www/cobbler/ks_mirror/BIOS/R710
cp /mnt/isolinux/SA* /var/www/cobbler/ks_mirror/BIOS/R720

将dtk导入到cobbler
#R610
cobbler distro add --name=R610_bios_upgrade --kernel=/var/www/cobbler/ks_mirror/BIOS/R610/SA.1  --initrd=/var/www/cobbler/ks_mirror/BIOS/R610/SA.2
cobbler profile add --name=R610_bios_upgrade --distro=R610_bios_upgrade

#R620
cobbler distro add --name=R620_bios_upgrade --kernel=/var/www/cobbler/ks_mirror/BIOS/R620/SA.1  --initrd=/var/www/cobbler/ks_mirror/BIOS/R620/SA.2
cobbler profile add --name=R620_bios_upgrade --distro=R620_bios_upgrade

#R710
cobbler distro add --name=R710_bios_upgrade --kernel=/var/www/cobbler/ks_mirror/BIOS/R710/SA.1  --initrd=/var/www/cobbler/ks_mirror/BIOS/R710/SA.2
cobbler profile add --name=R710_bios_upgrade --distro=R710_bios_upgrade

#R720
cobbler distro add --name=R720_bios_upgrade --kernel=/var/www/cobbler/ks_mirror/BIOS/R720/SA.1  --initrd=/var/www/cobbler/ks_mirror/BIOS/R720/SA.2
cobbler profile add --name=R720_bios_upgrade --distro=R720_bios_upgrade

#查看
cobbler profile list 
cobbler distro list
配置BIOS自动化升级脚本并与distro进行关联
mkdir -p  /var/lib/tftpboot/bios/R610
mkdir -p  /var/lib/tftpboot/bios/R620
mkdir -p  /var/lib/tftpboot/bios/R710
mkdir -p  /var/lib/tftpboot/bios/R720

Dell官网分别下载对应服务器型号的*.BIN格式的BIOS升级文件到以上对应目录中

#R610

[root@cobbler-9 bios]# cat R610/R610_bios_upgrade.sh 
#!/bin/bash

cat >>/tmp/r610_bios.sh <<EOF
tftp -g -r bios/R610/R610_BIOS_C6MRW_LN_6.4.0.BIN -l r610_bios.sh 192.168.1.1
sh r610_bios.sh -q 
sleep 10
reboot
EOF

sh /tmp/r610_bios.sh


参数-解释：
-g 表示下载文件
-r 表示远程文件名
-l 表示本地文件名
-p 表示上传文件
IP=tftp_server_ip

注：(1)如果不加上 -q(untended) 参数，则表示以交互式的方式进行，这明显不符合自动化需求。
    (2)若升级的固件版本与现有版本一致则跳过升级

#R620

[root@cobbler-9 bios]# cat R620/R620_bios_upgrade.sh 
#!/bin/bash

cat >>/tmp/r620_bios.sh <<EOF
tftp -g -r bios/R620/R620_BIOS_KR1XT_LN_2.5.4.BIN -l r620_bios.sh 192.168.1.1
sh r620_bios.sh -q 
sleep 10
reboot
EOF

sh /tmp/r620_bios.sh

#R710

[root@cobbler-9 bios]# cat R710/R710_bios_upgrade.sh 
#!/bin/bash

cat >>/tmp/r710_bios.sh <<EOF
tftp -g -r bios/R710/R710_BIOS_4HKX2_LN_6.4.0.BIN -l r710_bios.sh 192.168.1.1
sh r710_bios.sh -q 
sleep 10
reboot
EOF

sh /tmp/r710_bios.sh

#R720

[root@cobbler-9 bios]# cat R720/R720_bios_upgrade.sh 
#!/bin/bash

cat >>/tmp/r720_bios.sh <<EOF
tftp -g -r bios/R720/R720_BIOS_NPPKY_LN32_1.3.6.BIN -l r720_bios.sh 192.168.1.1
sh r720_bios.sh -q 
sleep 10
reboot
EOF

sh /tmp/r720_bios.sh

修改权限
chmod 755  R610/R610_bios_upgrade.sh 
chmod 755  R620/R620_bios_upgrade.sh 
chmod 755  R710/R710_bios_upgrade.sh 
chmod 755  R720/R720_bios_upgrade.sh 

#R610
cobbler distro edit --name=R610_bios_upgrade --kopts="share_type=tftp share_location=/bios/R610 share_script=R610_bios_upgrade.sh tftp_ip=192.168.1.1"

#R620
cobbler distro edit --name=R620_bios_upgrade --kopts="share_type=tftp share_location=/bios/R620 share_script=R620_bios_upgrade.sh tftp_ip=192.168.1.1"

#R710
cobbler distro edit --name=R710_bios_upgrade --kopts="share_type=tftp share_location=/bios/R710 share_script=R710_bios_upgrade.sh tftp_ip=192.168.1.1"

#R720
cobbler distro edit --name=R720_bios_upgrade --kopts="share_type=tftp share_location=/bios/R720 share_script=R720_bios_upgrade.sh tftp_ip=192.168.1.1"

保存配置并重启相关服务

cobbler sync

注意事项：

注：脚本中，如果不加上 -q(untended) 参数，则表示以交互式的方式进行，这明显不符合自动化需求。

关于BIOS降级：

如果需要降级，在脚本中sh后面需要加上 -f(force) 参数，同时要配合 -q 使用。

降级脚本示例如下：
[root@cobbler-9 bios]# cat R720/R720_bios_slow.sh 
#!/bin/bash
cat >>/tmp/r720_bios.sh <<EOF
tftp -g -r bios/R720/R720_BIOS_NPPKY_LN32_1.3.6.BIN -l r720_bios.sh 192.168.1.1
sh -f r720_bios.sh -q 
sleep 10
reboot
EOF

sh /tmp/r720_bios.sh

Done.
