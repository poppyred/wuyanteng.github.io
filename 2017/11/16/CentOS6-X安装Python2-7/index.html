CentOS6.X安装Python2.7

2017-11-16

升级Python到2.7

Centos 6.X自带Python 2.6，但是工作中可能为了满足需求而进行升级，一般的升级过程会导致很多问题，比如yum不能使用等等......而此方法不会导致问题。

安装阿里云的Yum源与Epel源

http://mirrors.aliyun.com/

#SCL安装Python
yum update
yum install centos-release-scl   
yum install python27

#启用新版本Python
scl enable python27 bash         

#安装额外的库
cd /opt/rh/python27/root/usr/bin/     #切换到Python安装目录
sudo LD_LIBRARY_PATH=$LD_LIBRARY_PATH ./easy_install-2.7 pip
sudo LD_LIBRARY_PATH=$LD_LIBRARY_PATH ./pip2.7 install requests

开机后自动使用Python2.7版本

echo "scl enable python27 bash" >>/etc/profile
source /etc/profile
