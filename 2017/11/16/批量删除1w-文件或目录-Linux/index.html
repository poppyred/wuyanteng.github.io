批量删除100万+文件

#cd到指定目录
cd /tmp
find ./ -type d -exec rm -rf {} \;   

提示“-bash: /bin/rm: Argument list too long”

正确姿势

cd /tmp

ls | xargs -n 50 rm -fr ls    

意思是列出当前目录下的所有文件或目录，然后以50个为一组的方式来删除。
其中这个值50可以根据实际情况调大或调小
