问题汇总-给openstack VM创建静态IP

2018-03-14 
由于业务需要, 需给openstack虚拟机添加静态IP地址，下面说说正确姿势。

1. 首先正常创建一台VM虚拟机(IP为自动分配), 查看其分配的IP地址
2. 在控制节点执行如下操作

2.1 查看所有运行的虚拟机网络信息
    neutron port-list
2.2 更新VM虚拟机的IP地址为静态

查看命令帮助    
neutron port-update --help  

命令格式：
neutron port-update --fixed-ip subnet_id=要修改的VM的SUBNET,ip_address=新IP  要修改的VM的id

修改示例:
neutron port-update --fixed-ip subnet_id=11ae5e0a-3f61-4ea0-870c-e75a279ce2b5,ip_address=192.168.56.100  7d813df1-d32a-4bb2-a217-df672d0daeae

2.3 需重启虚拟机即可生效 或者 修改vm系统内的IP地址为新修改的IP地址,然后重启网卡
