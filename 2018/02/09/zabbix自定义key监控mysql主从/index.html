zabbix自定义key监控mysql主从

2018-02-09 
以下通过自定义脚本对mysql主从存活状态以及io和sql线程进行监控

步骤1：创建脚本存放目录并使用脚本

mkdir -p  /var/lib/zabbix/scripts/mysql
在此目录使用以下脚本并赋予执行权限

注：以下脚本会生成3个文件

#!/bin/bash
USER=root
PASS=
io_check=`cat mysql_slave_status.log |awk "NR==1"|awk '{print $2}'`
sql_check=`cat mysql_slave_status.log |awk "NR==2"|awk '{print $2}'`

mysql -u${USER} -p${PASS} -e "show slave status\G;" |grep -i running > mysql_slave_status.log


#IO线程判断
if [ "${io_check}" = "Yes" ];then
   echo "1" > io_check.log
else
   echo "0" > io_check.log
fi


#sql线程判断
if [ "${sql_check}" = "Yes" ];then
   echo "1" > sql_check.log
else
   echo "0" > sql_check.log
fi

步骤2：配置crontab

#mysql slave自定义key
*/1 * * * * /bin/sh /var/lib/zabbix/scripts/mysql/slave_mysql.sh

步骤3: zabbix agentd.conf 添加自定义key, 如下：

UnsafeUserParameters=1      #启用自定义key

#mysql slave监控
UserParameter=io.slave,cat /var/lib/zabbix/scripts/mysql/io_check.log
UserParameter=sql.slave,cat /var/lib/zabbix/scripts/mysql/sql_check.log

#mysql
UserParameter=mysql.stat,ss -lntup|grep 3306|wc -l

步骤4：重启zabbix agent

步骤5：zabbix Web界面主机添加时，选择创建好的mysql模板即可

最后等待监控取值正常即可, 正常值为1
