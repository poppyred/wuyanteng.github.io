Shell变量自增

2018-02-14

变量自增的实现方法有很多种：

(1) i=`expr $i + 1`
(2) let i+=1
(3) ((i++))
(4) i=$[$i+1]
(5) i=$(( $i + 1 ))

expr方式变量自增-实例测试如下：

为了平台通用性建议使用expr方式

#!/bin/bash

num=0
while [ "$num" -lt 5 ]
do
  num=`expr $num + 1`
done

测试结果：

[root@localhost ~]# sh -x test.sh 
+ num=0
+ '[' 0 -lt 5 ']'
++ expr 0 + 1
+ num=1
+ '[' 1 -lt 5 ']'
++ expr 1 + 1
+ num=2
+ '[' 2 -lt 5 ']'
++ expr 2 + 1
+ num=3
+ '[' 3 -lt 5 ']'
++ expr 3 + 1
+ num=4
+ '[' 4 -lt 5 ']'
++ expr 4 + 1
+ num=5
+ '[' 5 -lt 5 ']'

let方式

#!/bin/bash

num=0

while [ "$num" -lt 5 ]
do
  let num+=1
done

测试结果:

[root@localhost ~]# sh -x test.sh 
+ num=0
+ '[' 0 -lt 5 ']'
+ let num+=1
+ '[' 1 -lt 5 ']'
+ let num+=1
+ '[' 2 -lt 5 ']'
+ let num+=1
+ '[' 3 -lt 5 ']'
+ let num+=1
+ '[' 4 -lt 5 ']'
+ let num+=1
+ '[' 5 -lt 5 ']'

((i++))方式

#!/bin/bash

num=0

while [ "$num" -lt 5 ]
do
  ((num++))
done

测试结果：

+ num=0
+ '[' 0 -lt 5 ']'
+ (( num++ ))
+ '[' 1 -lt 5 ']'
+ (( num++ ))
+ '[' 2 -lt 5 ']'
+ (( num++ ))
+ '[' 3 -lt 5 ']'
+ (( num++ ))
+ '[' 4 -lt 5 ']'
+ (( num++ ))
+ '[' 5 -lt 5 ']'
