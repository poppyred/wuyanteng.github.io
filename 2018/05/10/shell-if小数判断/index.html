shell if小数判断

2018-05-10 
由于程序需要，我要判断一个小数是否大于另一个值。即一个浮点数是否大于另一个浮点数。

加入$mya变量是一个浮点数，按照正常思路执行：

# if [ $mya -le 4 ]; then echo "ok";else echo "fail"; fi  
-bash: [: 5.7: integer expression expected  

shell报错：提示integer expression expected
意思就是shell默认只能判断整数，而不是浮点数

参考了大牛的方案，改为了如下：

# if [ $(echo "$mya <= 4"|bc) = 1 ]; then echo "ok";else echo "fail";fi  

测试：
# if [ $(echo "0.5 <= 4"|bc) = 1 ]; then echo "ok";else echo "fail";fi  
ok

# if [ $(echo "0.5 >= 4"|bc) = 1 ]; then echo "ok";else echo "fail";fi  
fail

# if [ $(echo "5.5 >= 4"|bc) = 1 ]; then echo "ok";else echo "fail";fi  
ok

# if [ $(echo "4.1 >= 4"|bc) = 1 ]; then echo "ok";else echo "fail";fi 
ok

接下来终于可以愉快的写脚本了...
